# Quiz App

Applicazione web per quiz a risposta multipla con sistema di autenticazione.

## Tecnologie
- Backend: Python Flask
- Database: SQLite
- Frontend: HTML, CSS, JavaScript

## Installazione

1. Clona il repository
2. Installa le dipendenze:
```bash
   pip install -r requirements.txt
```
3. Avvia il server:
```bash
   cd backend
   python app.py
```
4. Apri `frontend/login.html` nel browser

## Note
Progetto didattico - non utilizzare in produzione senza implementare le misure di sicurezza necessarie.
